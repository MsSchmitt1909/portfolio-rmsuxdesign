My Work

VocabQuest

VocabQuest is about an app that is making vocabulary learning fun for school children. This project showcases my initial design process from problem statements to the first iteration of a low-fidelity clickable prototype using Marvel.

WOD Hunt

WODHunt is a responsive web app for CrossFitters and functional fitness athletes who want to use the principles of scavenger hunts and geocaching for their training regime. This case study presents the entire design process, via user research and usability testing up to a high-fidelity mockup of the mobile app using mainly Adobe XD.
